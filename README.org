Welcome to Rust Monorepo Microservices, a framework designed to streamline the development of scalable microservice architectures in Rust using a monorepository approach.
#+end_quote

* Overview
This project aims to provide developers with a cohesive toolkit for managing multiple microservices within a single repository. By adopting a monorepo strategy, it simplifies code sharing, improves modularity, and enhances overall project maintainability. Rust's performance and safety features are leveraged to ensure robustness and efficiency in distributed systems.

* Features
+ *Monorepo Efficiency*: Manage multiple microservices in a unified repository.
+ *Simplified Development*: Streamline workflows with centralized code management.
+ *Enhanced Modularity*: Promote code reuse and maintainability across services.
+ *Dependency Management*: Simplify handling dependencies between microservices.
+ *Scalability*: Utilize Rust's performance advantages for scalable architectures.
+ *Safety*: Benefit from Rust's strong type system and memory safety guarantees.

Tuần 1: Rust cơ bản (Day 1–7)
Day 1: Cài đặt Rust, Cargo, Rust Analyzer
Cài đặt Rust:

Truy cập rust-lang.org và làm theo hướng dẫn để cài đặt Rust bằng rustup.
Cài đặt Cargo:

Cargo sẽ được cài đặt tự động cùng với Rust.
Cài đặt Rust Analyzer:

Cài đặt Rust Analyzer qua Visual Studio Code hoặc theo hướng dẫn trên trang chính thức.
Day 2: Ownership, Borrowing, Lifetimes
Ownership:

Tìm hiểu về nguyên tắc ownership trong Rust. Có 3 quy tắc cơ bản:
Mỗi giá trị trong Rust đều có một owner.
Chỉ có thể có một owner tại một thời điểm.
Khi owner ra khỏi phạm vi, giá trị sẽ bị giải phóng.
Borrowing:

Tìm hiểu cách mượn giá trị bằng cách sử dụng tham chiếu (&).
Lifetimes:

Nắm rõ khái niệm về lifetimes để đảm bảo an toàn bộ nhớ.
Bài tập: Viết hàm tính toán, quản lý bộ nhớ
Viết một hàm tính tổng của một mảng số nguyên, đảm bảo rằng bạn sử dụng ownership và borrowing đúng cách.
Day 3: Struct, Enum, Traits
Struct:

Tìm hiểu cách định nghĩa và sử dụng struct.
Enum:

Tìm hiểu cách định nghĩa và sử dụng enum.
Day 4: Error handling (Result, Option)
Tìm hiểu cách xử lý lỗi trong Rust bằng Result và Option.
Bài tập: Tạo struct User, validate dữ liệu
Tạo một struct User với các trường như tên và tuổi. Viết hàm để xác thực dữ liệu của user.

Tìm hiểu về các collection như Vec và HashMap.
Day 6: Pattern Matching
